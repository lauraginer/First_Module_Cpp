--Clase ifstream y ofstream:

ifstream:

La clase ifstream en C++ (Input File Stream) se utiliza para leer datos desde archivos,
siendo parte de la cabecera <fstream> y derivando de istream, lo que permite usar 
operadores como >> y getline() para leer texto línea por línea o palabra por palabra, 
abriendo archivos pasándoles el nombre al constructor o usando el método open(), 
y siempre se debe verificar si el archivo se abrió correctamente y cerrarlo al finalizar.

Características Clave
Propósito: Leer contenido de archivos.
Cabecera: Se incluye #include <fstream> para usarla.
Jerarquía: Es una clase de flujo de entrada para archivos (input file stream), derivada de istream, similar a cin.
Operaciones: Permite leer datos usando el operador de extracción >> o la función std::getline() para leer líneas completas.


ofstream:

La clase ofstream en C++ (parte de <fstream>) se usa para escribir datos en archivos (salida de archivos), 
creando o abriendo un archivo para escritura y permitiendo usar operadores de flujo (<<) como con cout. 
Se crea un objeto ofstream, opcionalmente pasándole el nombre del archivo al constructor, y luego se usa 
para volcar información al disco, cerrando el archivo al terminar.

Características clave
Propósito: Escritura de datos en archivos de texto o binarios.
Herencia: Deriva de ostream, heredando su funcionalidad de salida.
Encabezado: Requiere #include <fstream>.
Modo de apertura (por defecto): Crea el archivo o lo trunca (borra el contenido) si ya existe (modo ios::out).
